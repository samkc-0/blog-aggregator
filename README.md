# blog-aggregator

## Dependencies

postgres, go

install with:
git clone ...
cd rss-aggregator

## usage

`rss-aggregator <command>`

register <username>
agg <time_between_requests>
feeds
unfollow <url>
follow <url>
following
browse [limit,]
login <username>
users
reset
addfeed <title> <url>

Note: <time_between_requests> is a duration expressed as a string. For example, 1m is one minute, 30s is thirty seconds.
